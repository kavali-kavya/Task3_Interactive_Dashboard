# Task 3 – Interactive Dashboard using Power BI

## Project Overview

This project demonstrates the creation of an interactive sales dashboard using Microsoft Power BI. The dashboard provides meaningful business insights through interactive visualizations and KPI cards.

## Features

* KPI Cards (Total Sales, Total Orders, Average Sales, Total Customers)
* Sales by Product (Bar Chart)
* Sales by Category (Column Chart)
* Sales by City (Pie Chart)
* Monthly Sales Trend (Line Chart)
* Age vs Total Sales (Scatter Plot)
* Interactive Slicers (Category, City, Gender)

## Tools Used

* Microsoft Power BI
* Microsoft Excel

## Files Included

* Task3_Interactive_Dashboard.pbix
* Dashboard_Screenshot.png

## Outcome

The dashboard enables users to analyze sales performance, identify trends, and explore data interactively using filters and visualizations.
